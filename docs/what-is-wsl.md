# What is WSL
Windows Subsystem for Linux (WSL) lets user run a fully functional and native GNU/Linux environment directly on Windows without using a virtual machine or dual-booting the system.

WSL allows you to never switch OS's but still have the linux experience inside your main OS , it saves the hastle of [Dual Booting](#dual-booting) or spining up a [Virtual Machine](#virtual-machine) On your main OS

### Dual Booting
Installing two different operating systems (such as Windows and Linux) on the same computer's SSD/HardDrive
### Virtual Machine
Virtual Machine (VM) is a software-based computer that runs inside a physical computer (host).
